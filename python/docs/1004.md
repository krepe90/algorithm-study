## 1002. [어린 왕자](https://www.acmicpc.net/problem/1004)

### Problem analysis
시작점, 끝점 각각 몇개의 원이 감싸고 있는지 확인하고, 이중 중복된걸 제거해서 고르면 될 것 같다.

### Solution 1
1. 첫번째 입력받은 숫자만큼 코드를 반복(반복문)
2. 두번째에 입력받는 숫자들 4개가 시작점, 끝점 `(x1, y1), (x2, y2)`
3. 세번째로 입력받는 숫자가 행성계의 숫자. 그만큼 또 반복
4. 행성의 `(x, y, r)` 값을 받으면 `check(point, planet)` 함수를 이용하여 시작점/끝점이 해당 행성계 내에 있는지 확인 후 저장
5. XOR 연산으로 둘중 하나만 해당 행성계에 포함되어있으면 `po`변수에 +1
6. `1`번 반복문의 마지막마다 `ret`변수에 `po`값을 추가
7. `ret`변수에 있는 결과값들 순서대로 출력
