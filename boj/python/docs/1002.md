## 1002. [터렛](https://www.acmicpc.net/problem/1002)

### Problem analysis
좌표평면 두 원(터렛)의 좌표와 반지름을 각각 주고, 이들이 몇번 겹치는지 구하는 문제이다.

### Solution 1
1. `두 원의 중심끼리의 거리`(이하 `d`)를 피타고라스 법칙을 이용하여 구한다.
2. 두 원이 거리가 충분히 떨어져있는지 구한다. `d`가 `r1`과 `r2`보다 작은지 큰지로 구분.
3. 두 원이 떨어져있으면 `r1`과 `r2`를 더하여, 위에서 구한 `d`와 비교한다.
   - `r1 + r2 > d` 인 경우: 두 점에서 만남
   - `r1 + r2 == d` 인 경우: 한 점에서 만남
   - `r1 + r2 < d` 인 경우: 만나지 않음
   - 두 원이 같은 경우: 무한대
4. 그렇지 않으면 `r1`과 `r2`를 빼서, 위에서 구한 `d`와 비교한다.
   - `|r1 - r2| < d` 인 경우: 두 점에서 만남
   - `|r1 - r2| == d` 인 경우: 한 점에서 만남
   - `|r1 - r2| > d` 인 경우: 만나지 않음
   - 두 원이 같은 경우: 무한대
5. 따라서 위 경우에 각각 `2`, `1`, `0`, `-1`을 출력해주면 된다.
6. 이를 test case 만큼 반복해야 하기 때문에 `ret`에 값들을 `list`형으로 저장한 후 반복문으로 출력한다.
